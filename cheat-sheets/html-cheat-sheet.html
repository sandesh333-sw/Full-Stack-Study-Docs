Quick reference for HTML elements and attributes

Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <!-- Content here -->
</body>
</html>

Text Elements

<h1> to <h6> Headings (largest to smallest) <p> Paragraph <br> Line break <hr> Horizontal rule <strong> Important text (bold) <em> Emphasized text (italic) <mark> Highlighted text <small> Smaller text <del> Deleted text <ins> Inserted text

Lists

Unordered List:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Ordered List:

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

Description List:

<dl>
    <dt>Term</dt>
    <dd>Definition</dd>
</dl>

Links & Images

Links:

<a href="url">Link text</a>
<a href="mailto:email@example.com">Email</a>
<a href="tel:+1234567890">Phone</a>
<a href="#section">Internal link</a>

Images:

<img src="image.jpg" alt="Description">
<img src="image.jpg" alt="Desc" width="300">

Figure:

<figure>
    <img src="image.jpg" alt="Chart">
    <figcaption>Caption</figcaption>
</figure>

Tables

<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
</table>
colspan="n" Span across columns rowspan="n" Span across rows

Forms

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <button type="submit">Submit</button>
</form>

Input Types

type="text" Text input type="email" Email input type="password" Password input type="number" Number input type="date" Date picker type="checkbox" Checkbox type="radio" Radio button type="file" File upload type="submit" Submit button type="reset" Reset button

Semantic HTML5

<header> Page/section header <nav> Navigation links <main> Main content <article> Independent content <section> Content section <aside> Sidebar content <footer> Page/section footer <time> Date/time <address> Contact information

Media Elements

Video:

<video controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
</video>

Audio:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
</audio>

Meta Tags

<meta charset="UTF-8">
<meta name="viewport" 
      content="width=device-width, initial-scale=1.0">
<meta name="description" 
      content="Page description">
<meta name="keywords" 
      content="html, css, javascript">
<meta name="author" content="Your Name">

Global Attributes

id="unique" Unique identifier class="name" CSS class name(s) style="css" Inline CSS styles title="text" Tooltip text lang="en" Language code data-* Custom data attributes hidden Hide element tabindex="n" Tab order
